      Halo teman-teman. di tugas informatika ini,
      kami akan membuat game quiz matematika
      untuk anak sd. pertanyaannya tidak terlalu
      sulit jadi anak sd pasti bisa
      mengerjakannya. anggota kelompok kami:
      
      1. Hasbiallah (16) sebagai Web programmer
      pekerjaannya adalah pengembangan dan 
      pemeliharaan situs web, membuat aplikasi 
      web interaktif, mengimplementasikan desain
      yang diberikan ke dalam kode.

      2. Fauzan (11) sebagai Application
      pekerjaannya adalah merancang, mengembangkan, 
      dan memelihara aplikasi perangkat lunak untuk 
      berbagai platform. dia membuat program Scratch
      yang akan dipakai untuk game kami. sebagai
      application, dia bekerja sama dengan 
      graphic designer untuk membuat gamenya.
      
      3. Gerrald (12) sebagai Graphic designer
      pekerjaannya adalah merancang grafik dan
      elemen visual untuk berbagai platform seperti
      situs web, aplikasi mobile, dan identitas 
      merek. singkatnya, dia mencari design yang 
      akan dipakai di game kami. seperti membuat
      gambar karakter game, background game, dan 
      hiasan-hiasan di dalam game lainnya.
      
      4. Fadel (9) sebagai Content writer
      pekerjaannya adalah penelitian topik yang
      relevan, pengembangan ide, penulisan teks 
      yang menarik, dan pengeditan untuk memastikan
      kualitasnya.
