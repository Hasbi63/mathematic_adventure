      Halo teman-teman. di tugas informatika ini,
      kami akan membuat game quiz matematika
      untuk anak sd. pertanyaannya tidak terlalu
      sulit jadi anak sd pasti bisa
      mengerjakannya.
      
      alasan kami membuat game ini karena beberapa
      siswa sd terkadang tidak mengerti matematika,
      jadi kami bertujuan agar anak anak tersebut
      menjadi lebih paham matematika.
      
      "Mathematics Adventure" diambil dari dua kata 
      yaitu matematika dan petualangan. harapan kami 
      adalah anak anak bisa menikmati game kami 
      dan mereka akan beranggapan sedang berpetualang
      bukannya belajar.

      tugas progammer antara lain:
      1. konten writer: penelitian topik yang relevan,
      pengembangan ide, penulisan teks yang menarik, 
      dan pengeditan untuk memastikan kualitasnya.

      2. web programer:pengembangan dan pemeliharaan
      situs web, membuat aplikasi web interaktif, 
      mengimplementasikan desain yang diberikan ke 
      dalam kode.

      3. grafik designer: merancang grafik dan
      elemen visual untuk berbagai platform seperti
      situs web, aplikasi mobile, materi pemasaran, 
      dan identitas merek.

      4. aplikasi programer: merancang, mengembangkan, 
      dan memelihara aplikasi perangkat lunak untuk 
      berbagai platform.
      
      dari tugas tugas tersebut, tugas anggota kelompok
      kami adalah:
      1. Fadel (9) sebagai Konten writer, dia bertugas
      sebagai orang yang menulis teks dalam html dan
      memberi ide ide.

      2. Hasbiallah (16) sebagai Web progamer
